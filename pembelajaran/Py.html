Pembelajaran Python

Dasar-dasar Python untuk pemula hingga mahir

Apa itu Python?

Python adalah bahasa pemrograman tingkat tinggi yang dikembangkan oleh Guido van Rossum dan dirilis pertama kali pada tahun 1991. Python dirancang untuk mudah dipahami, bersifat open-source, dan memiliki komunitas yang besar.

Python digunakan di berbagai bidang seperti pengembangan web (Django, Flask), kecerdasan buatan (TensorFlow, PyTorch), analisis data (Pandas, NumPy), automasi, hingga pengembangan aplikasi desktop.

Variabel & Tipe Data

Dalam Python, tidak perlu mendeklarasikan tipe variabel secara eksplisit karena Python adalah bahasa yang dinamis. Interpreter akan mengenali tipe data berdasarkan nilai yang diberikan. 

# Contoh:
nama = "Budi"         # string
umur = 25             # integer
tinggi = 170.5        # float
is_aktif = True       # boolean

Operasi Aritmatika

Python mendukung operasi matematika dasar seperti penjumlahan, pengurangan, perkalian, pembagian, modulus, dan perpangkatan. 

a = 10
b = 3
print(a + b)   # 13
print(a - b)   # 7
print(a * b)   # 30
print(a / b)   # 3.33
print(a % b)   # 1
print(a ** b)  # 1000

Fungsi (Function)

Fungsi digunakan untuk membuat kode yang terstruktur dan bisa digunakan kembali. Gunakan kata kunci def untuk membuat fungsi. 

def sapa(nama):
    return "Halo, " + nama

print(sapa("Dina"))

Fungsi juga bisa menerima lebih dari satu argumen dan mengembalikan nilai kompleks.

Kondisi (if, elif, else)

Gunakan pernyataan kondisi untuk membuat keputusan dalam program. 

nilai = 85
if nilai >= 90:
    print("A")
elif nilai >= 80:
    print("B")
else:
    print("C")

Indentasi (spasi/tab) penting dalam Python karena menentukan blok kode.

Perulangan

Python memiliki dua jenis loop utama: for dan while.

# For loop
for i in range(1, 6):
    print("Angka ke-", i)

# While loop
x = 1
while x <= 5:
    print(x)
    x += 1

Gunakan break untuk menghentikan loop dan continue untuk melewati iterasi.

List & Dictionary

List digunakan untuk menyimpan kumpulan data dalam satu variabel. Dictionary menyimpan data dengan pasangan kunci-nilai. 

# List
buah = ["Apel", "Jeruk", "Mangga"]
print(buah[0])  # Apel

# Dictionary
orang = {
    "nama": "Rina",
    "umur": 20
}
print(orang["nama"])  # Rina

Input & Output

Gunakan input() untuk menerima data dari user dan print() untuk menampilkannya ke layar. 

nama = input("Siapa nama kamu? ")
print("Halo,", nama)

Penggunaan Modul

Python memiliki ribuan modul bawaan dan eksternal. Gunakan import untuk mengakses modul. 

import math

print(math.sqrt(16))   # 4.0
print(math.pi)         # 3.141592653589793

Exception Handling

Python menyediakan mekanisme penanganan error dengan try dan except. 

try:
    x = int(input("Masukkan angka: "))
    hasil = 10 / x
    print("Hasil:", hasil)
except ZeroDivisionError:
    print("Tidak bisa dibagi dengan nol.")
except ValueError:
    print("Input harus berupa angka.")

Latihan Mini Proyek

Membuat kalkulator luas segitiga:

def luas_segitiga(alas, tinggi):
    return 0.5 * alas * tinggi

a = float(input("Masukkan alas: "))
t = float(input("Masukkan tinggi: "))
print("Luas:", luas_segitiga(a, t))
