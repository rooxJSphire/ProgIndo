Dasar Pemrograman Java untuk Pemula

Apa itu Java?

Java adalah bahasa pemrograman berorientasi objek yang populer untuk pengembangan aplikasi desktop, mobile (terutama Android), dan web. Java bersifat portable karena berjalan di atas Java Virtual Machine (JVM).

Struktur Dasar Java

public class HelloWorld {
  public static void main(String[] args) {
    System.out.println("Halo Dunia!");
  }
}

Setiap program Java dimulai dari method main() di dalam sebuah class.

Variabel dan Tipe Data

String nama = "Budi";
int umur = 20;
double tinggi = 170.5;
boolean isAktif = true;
  • Java menggunakan tipe data seperti int, double, boolean, dan String.
  • Setiap variabel harus memiliki tipe data yang jelas.

Kondisional

int nilai = 85;
if (nilai >= 80) {
  System.out.println("Nilai Anda A");
} else {
  System.out.println("Nilai Anda B");
}

Java mendukung struktur kontrol if, else, dan switch.

Perulangan (Looping)

for (int i = 1; i <= 5; i++) {
  System.out.println("Baris ke-" + i);
}

Java memiliki loop seperti for, while, dan do-while.

Fungsi (Method)

public static String salam(String nama) {
  return "Halo, " + nama + "!";
}

System.out.println(salam("Dina"));

Method di Java digunakan untuk menyimpan logika yang dapat digunakan kembali.

Class dan Object

public class Mobil {
  String merk = "Toyota";

  void jalan() {
    System.out.println("Mobil berjalan...");
  }
}

public class Main {
  public static void main(String[] args) {
    Mobil m = new Mobil();
    m.jalan();
  }
}

Java adalah bahasa berorientasi objek. Segala sesuatu dibangun dari class dan object.

Array

String[] buah = {"Apel", "Jeruk", "Mangga"};
System.out.println(buah[0]);

Array di Java menyimpan kumpulan data bertipe sama dalam satu variabel.

Input dari User

import java.util.Scanner;

Scanner input = new Scanner(System.in);
System.out.print("Masukkan nama: ");
String nama = input.nextLine();
System.out.println("Halo, " + nama);

Gunakan class Scanner untuk membaca input dari user di terminal.

Exception Handling (Try-Catch)

try {
  int hasil = 10 / 0;
} catch (ArithmeticException e) {
  System.out.println("Terjadi kesalahan: " + e.getMessage());
}

Gunakan try-catch untuk menangani error saat runtime.

File I/O Sederhana

import java.io.FileWriter;
import java.io.IOException;

public class FileDemo {
  public static void main(String[] args) {
    try {
      FileWriter writer = new FileWriter("data.txt");
      writer.write("Halo, Java File!");
      writer.close();
    } catch (IOException e) {
      System.out.println("Gagal menulis file.");
    }
  }
}

Java mendukung operasi file menggunakan class seperti FileWriter dan BufferedReader.
