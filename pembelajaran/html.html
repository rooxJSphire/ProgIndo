Belajar HTML Dasar

Pahami fondasi pembuatan website dari awal.

Apa itu HTML?

HTML (HyperText Markup Language) adalah bahasa markup yang digunakan untuk membuat struktur halaman web. HTML bukan bahasa pemrograman, melainkan bahasa penanda (markup).

Struktur Dasar HTML

<!DOCTYPE html>
<html>
  <head>
    <title>Judul Halaman</title>
  </head>
  <body>
    <h1>Halo Dunia!</h1>
    <p>Ini paragraf pertama saya.</p>
  </body>
</html>

Penjelasan:

  • <!DOCTYPE html>: Deklarasi HTML5.
  • <html>: Root dari semua elemen HTML.
  • <head>: Informasi metadata dan judul.
  • <body>: Konten utama yang ditampilkan.

Heading dan Paragraf

<h1>Heading 1</h1>
<h2>Heading 2</h2>
<p>Ini adalah paragraf.</p>

Gunakan h1 hingga h6 untuk judul. p untuk paragraf biasa.

Menambahkan Gambar

<img src="gambar.jpg" alt="Deskripsi gambar">

src menentukan sumber gambar, alt untuk deskripsi alternatif.

Menambahkan Tautan

<a href="https://google.com">Kunjungi Google</a>

Tautan akan membawa pengguna ke halaman lain atau URL eksternal.

Daftar List

<ul>
  <li>Item 1</li>
  <li>Item 2</li>
</ul>

<ol>
  <li>Item 1</li>
  <li>Item 2</li>
</ol>

ul untuk daftar tidak berurutan, ol untuk daftar bernomor.

Membuat Tabel

<table border="1">
  <tr>
    <th>Nama</th>
    <th>>Umur</th>
  </tr>
  <tr>
    <td>Ali</td>
    <td>21</td>
  </tr>
</table>

Tabel digunakan untuk menampilkan data dalam bentuk baris dan kolom.

Formulir Input

<form action="/submit" method="post">
  <label for="nama">Nama:</label>
  <input type="text" id="nama" name="nama">
  <input type="submit" value="Kirim">
</form>

Form digunakan untuk mengambil input dari pengguna seperti nama, email, dll.

Komentar dalam HTML

<!-- Ini adalah komentar -->

Komentar tidak akan ditampilkan di browser, hanya terlihat di kode sumber.

Kesimpulan

HTML adalah fondasi utama pembuatan website. Jika kamu sudah paham HTML, kamu bisa lanjut ke CSS dan JavaScript untuk memperindah dan menambah interaktivitas website-mu.
