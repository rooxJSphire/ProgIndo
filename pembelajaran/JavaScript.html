Pembelajaran JavaScript

Dasar-dasar JavaScript untuk pemula hingga mahir

Apa itu JavaScript?

JavaScript adalah bahasa pemrograman yang digunakan untuk membuat situs web lebih interaktif. JavaScript dapat digunakan untuk mengontrol elemen HTML, merespon event (seperti klik atau hover), dan membangun aplikasi web dinamis.

Variabel & Tipe Data

JavaScript memiliki tiga cara mendeklarasikan variabel:

  • var – versi lama, cakupan fungsi
  • let – untuk variabel lokal (modern)
  • const – untuk konstanta (tidak bisa diubah)

// Contoh:
let nama = "Ali";
const umur = 21;
var status = "Mahasiswa";

Fungsi (Function)

Fungsi digunakan untuk menyimpan logika agar dapat digunakan berulang.


function sapa(nama) {
  return "Halo, " + nama + "!";
}
console.log(sapa("Dina")); // Halo, Dina!

Kondisi (if, else)


let nilai = 85;
if (nilai >= 80) {
  console.log("Lulus dengan baik");
} else {
  console.log("Perlu belajar lagi");
}

Perulangan

Perulangan digunakan untuk mengeksekusi kode berulang-ulang.


// For loop
for (let i = 1; i <= 5; i++) {
  console.log("Angka ke-" + i);
}

Array & Object


// Array
let buah = ["Apel", "Pisang", "Jeruk"];
console.log(buah[0]); // Apel

// Object
let orang = {
  nama: "Fahmi",
  umur: 22
};
console.log(orang.nama); // Fahmi

Event Listener

Digunakan untuk mendeteksi interaksi user:


document.getElementById("klikSaya").addEventListener("click", function() {
  alert("Tombol diklik!");
});

DOM Manipulation

Untuk memodifikasi isi HTML dengan JavaScript:


document.getElementById("judul").innerText = "Judul Baru";

Latihan Mini Proyek

Membuat tombol interaktif:








function hitung() {
  let hasil = 5 * 10;
  document.getElementById("hasil").innerText = "Hasil: " + hasil;
}
