Dasar Pemrograman PHP untuk Pemula

Apa itu PHP?

PHP (Hypertext Preprocessor) adalah bahasa scripting server-side yang digunakan untuk membuat halaman web dinamis. PHP biasanya digunakan bersama HTML dan database seperti MySQL.

Struktur Dasar PHP

<?php
echo "Halo Dunia!";
?>

Semua kode PHP dibuka dengan <?php dan ditutup dengan ?>.

Variabel dan Tipe Data

<?php
$nama = "Budi";
$umur = 20;
echo "Nama saya $nama dan umur saya $umur tahun.";
?>
  • Variabel diawali dengan simbol $
  • Tipe data bisa berupa string, integer, boolean, array, dll

Kondisional

<?php
$nilai = 85;
if ($nilai >= 80) {
    echo "Nilai Anda A";
} else {
    echo "Nilai Anda B";
}
?>

PHP mendukung struktur kontrol seperti if, else, dan switch.

Perulangan (Looping)

<?php
for ($i = 1; $i <= 5; $i++) {
    echo "Baris ke-$i <br>";
}
?>

PHP memiliki for, while, dan foreach loop.

Fungsi

<?php
function salam($nama) {
    return "Halo, $nama!";
}
echo salam("Dina");
?>

Fungsi digunakan untuk menyimpan blok kode yang bisa digunakan kembali.

Form HTML dan PHP

<form method="post">
  Nama: <input type="text" name="nama">
  <input type="submit" value="Kirim">
</form>

<?php
if ($_SERVER["REQUEST_METHOD"] == "POST") {
  $nama = $_POST["nama"];
  echo "Halo, $nama!";
}
?>

PHP dapat menangani input dari form HTML menggunakan metode GET atau POST.

Koneksi ke MySQL

<?php
$conn = mysqli_connect("localhost", "root", "", "belajar");
if (!$conn) {
  die("Koneksi gagal: " . mysqli_connect_error());
}
echo "Koneksi berhasil!";
?>

Koneksi database menggunakan fungsi mysqli_connect.
