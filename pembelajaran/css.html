Belajar CSS (Cascading Style Sheets)

Dasar-dasar CSS untuk mempercantik tampilan halaman web

Apa Itu CSS?

CSS (Cascading Style Sheets) adalah bahasa desain yang digunakan untuk mengatur tampilan dan layout halaman HTML. Dengan CSS, kita bisa mengatur warna, ukuran, font, posisi elemen, dan animasi.

Struktur Dasar CSS

selector {
  property: value;
}

Contoh penggunaan: 

h1 {
  color: blue;
  text-align: center;
}

Penjelasan: Selector `h1` akan memiliki teks berwarna biru dan berada di tengah.

Cara Menghubungkan CSS ke HTML

  1. Internal CSS: Ditulis di dalam tag <style> dalam file HTML.
  2. External CSS: Disimpan dalam file .css lalu di-link lewat <link>.
  3. Inline CSS: Ditulis langsung dalam elemen HTML.
<link rel="stylesheet" href="style.css">

Contoh Styling

body {
  background-color: #f5f5f5;
  font-family: Arial, sans-serif;
}

button {
  background: linear-gradient(to right, #00c6ff, #0072ff);
  color: white;
  padding: 10px 20px;
  border: none;
  border-radius: 10px;
  transition: 0.3s;
}

button:hover {
  transform: scale(1.1);
  box-shadow: 0 4px 10px rgba(0,0,0,0.2);
}

Efek: Button akan berubah ukuran dan memiliki efek glow saat hover.

Selector CSS

  • .class - Memilih elemen berdasarkan class
  • #id - Memilih elemen berdasarkan ID
  • * - Memilih semua elemen
  • tag - Memilih semua tag tertentu seperti `p`, `div`, `h1`

Box Model

Setiap elemen HTML adalah sebuah kotak yang terdiri dari:

  • Content - Isi dari elemen
  • Padding - Ruang antara konten dan border
  • Border - Garis tepi elemen
  • Margin - Ruang luar antar elemen

Responsive Design

Gunakan media query untuk membuat tampilan menyesuaikan ukuran layar:

@media (max-width: 768px) {
  body {
    font-size: 14px;
  }
}

Animasi CSS

@keyframes fadeIn {
  0% { opacity: 0; }
  100% { opacity: 1; }
}

.fade {
  animation: fadeIn 1s ease-in-out;
}

Penjelasan: Elemen dengan class `.fade` akan muncul perlahan.

Tips Best Practice

  • Gunakan class untuk styling, hindari inline style.
  • Susun CSS agar mudah dibaca (indentasi, grup kode).
  • Pakai tool seperti VS Code + Live Server.
  • Manfaatkan devtools browser untuk debugging.

Latihan CSS
