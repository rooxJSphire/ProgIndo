Pembelajaran Go

Belajar bahasa pemrograman Go dari dasar

Apa itu Go?

Go (atau Golang) adalah bahasa pemrograman yang dikembangkan oleh Google. Go dirancang untuk efisiensi, kecepatan, dan kemudahan dalam pengembangan perangkat lunak skala besar.

Struktur Dasar Program Go

package main
import "fmt"

func main() {
    fmt.Println("Halo, dunia!")
}

Program dasar di atas menampilkan teks ke layar menggunakan fungsi fmt.Println.

Tipe Data

  • int - bilangan bulat
  • float64 - bilangan desimal
  • string - teks atau kumpulan karakter
  • bool - nilai benar atau salah

Variabel dan Konstanta

var nama string = "Andi"
umur := 20
const pi = 3.14

Go menggunakan var untuk deklarasi variabel dan := untuk deklarasi cepat. Konstanta ditentukan dengan const.

Kondisi dan Perulangan

angka := 10
if angka > 5 {
    fmt.Println("Lebih besar dari 5")
}

for i := 0; i < 3; i++ {
    fmt.Println("Perulangan ke-", i)
}

Fungsi dalam Go

func tambah(a int, b int) int {
    return a + b
}

func main() {
    fmt.Println(tambah(2, 3))
}

Array dan Slice

var angka = [3]int{1, 2, 3}
var nama = []string{"A", "B", "C"}

fmt.Println(angka)
fmt.Println(nama)

Array memiliki ukuran tetap, sedangkan slice fleksibel dan lebih sering digunakan dalam Go.

Struct dan Method

type Mahasiswa struct {
    Nama string
    Umur int
}

func (m Mahasiswa) Perkenalan() {
    fmt.Println("Halo, saya", m.Nama)
}

Struct digunakan untuk membuat tipe data buatan dan method ditambahkan untuk perilaku objek.

Package dan Import

import (
    "fmt"
    "math"
)

func main() {
    fmt.Println(math.Sqrt(16))
}

Go menggunakan import untuk menggunakan library lain seperti fmt dan math.
