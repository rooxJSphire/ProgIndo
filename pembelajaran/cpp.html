Pembelajaran C++

Belajar bahasa pemrograman C++ dari dasar

Apa itu C++?

C++ adalah bahasa pemrograman yang dikembangkan dari bahasa C. C++ mendukung paradigma pemrograman prosedural dan berorientasi objek, serta banyak digunakan untuk pengembangan sistem, game, dan aplikasi performa tinggi.

Struktur Dasar Program C++

#include <iostream>
using namespace std;

int main() {
    cout << "Halo, dunia!" << endl;
    return 0;
}

Program dasar di atas mencetak teks ke layar menggunakan cout dari library <iostream>.

Tipe Data

  • int - bilangan bulat
  • float / double - bilangan desimal
  • char - karakter tunggal
  • string - teks atau kumpulan karakter
  • bool - nilai benar atau salah

Variabel dan Konstanta

#include <iostream>
using namespace std;

int main() {
    int umur = 20;
    const float pi = 3.14;
    string nama = "Andi";
    cout << nama << " berusia " << umur << " tahun." << endl;
}

Variabel dideklarasikan dengan tipe data tertentu. Konstanta menggunakan keyword const.

Kondisi dan Perulangan

int angka = 10;
if (angka > 5) {
    cout << "Lebih besar dari 5" << endl;
}

for (int i = 0; i < 3; i++) {
    cout << "Perulangan ke-" << i << endl;
}

Fungsi dalam C++

int tambah(int a, int b) {
    return a + b;
}

int main() {
    cout << tambah(2, 3) << endl;
    return 0;
}

Fungsi digunakan untuk memisahkan logika dan membuat kode lebih terstruktur dan dapat digunakan kembali.

Array dan Vector

#include <iostream>
#include <vector>
using namespace std;

int main() {
    int angka[3] = {1, 2, 3};
    vector<string> nama = {"A", "B", "C"};

    cout << angka[0] << endl;
    cout << nama[1] << endl;
}

Array memiliki ukuran tetap, sedangkan vector dari STL bersifat fleksibel dan dinamis.

Struct dan Method

struct Mahasiswa {
    string nama;
    int umur;

    void perkenalan() {
        cout << "Halo, saya " << nama << endl;
    }
};

int main() {
    Mahasiswa m;
    m.nama = "Budi";
    m.umur = 21;
    m.perkenalan();
}

Struct di C++ digunakan untuk membuat tipe data khusus, dan bisa memiliki method (fungsi anggota).

Input dari User

#include <iostream>
using namespace std;

int main() {
    string nama;
    cout << "Masukkan nama: ";
    cin >> nama;
    cout << "Halo, " << nama << "!" << endl;
}

Gunakan cin untuk menerima input dari pengguna di terminal.

File I/O Dasar

#include <iostream>
#include <fstream>
using namespace std;

int main() {
    ofstream file("data.txt");
    file << "Ini data dari program C++";
    file.close();
}

C++ mendukung file handling menggunakan fstream seperti ofstream dan ifstream.
